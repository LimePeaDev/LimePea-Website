📖 Help & Commands

⚙️ Admin & Moderation

  • lp!sync — Sync slash commands.
  • lp!warn <user> [reason] — Warn a user.
  • lp!kick <user> [reason] — Kick a member.
  • lp!ban <user> [reason] — Ban a user.
  • lp!unban <user_id> — Unban someone by ID.
  • lp!unwarn <user> — Remove the latest warning.
  • lp!warnings <user> — Show all warnings for a user.
  • lp!silence <user> <time> — Timeout a member (e.g., 10m, 1h).
  • lp!unsilence <user> — Remove timeout from a member.
  • lp!setwelcome <#channel> — Set the welcome message channel.
  • lp!setgoodbye <#channel> — Set the goodbye message channel.
  • lp!setautorole <role> — Set autorole for new members.
  • lp!autorole — View the current autorole.
  • lp!testwelcome — Simulate the welcome message.
  • lp!testgoodbye — Simulate the goodbye message.
  • lp!addreactionrole <message_id> <emoji> <role> — Add a reaction role.
  • lp!removereactionrole <message_id> <emoji> — Remove a reaction role.
  • lp!setlevelrole <level> <role> — Assign a role when users level up.
  • lp!removelevelrole <level> — Remove a level-up role.
  • lp!listlevelroles — Show all level role rewards.
  • lp!nickunlock <@user> — Allow an admin to force change a users nickname.
  • lp!autoroletest — Test if autorole works correctly.
  • lp!serverlock — Lock the entire server.
  • lp!serverunlock — Unlock the entire server.
  • lp!lock <#channel> — Lock a channel.
  • lp!unlock <#channel> — Unlock a channel.
  • lp!slowmode <#channel> <seconds> — Set channel slowmode.
  • lp!tag <name> — View a saved tag by name.
  • lp!tags — List all saved tags.
  • lp!tag <name> <content> — Create or update a tag.
  • lp!untag <name> — Delete a tag.
  • lp!purge <number> — Bulk delete messages.

💸 Economy & Stock

  • lp!balance — View your lime balance.
  • lp!daily — Collect daily lime rewards.
  • lp!gamble <amount> — Gamble your limes.
  • lp!slots <amount> — Spin the slot machine.
  • lp!coinflip <amount> — Flip a coin and win or lose limes.
  • lp!job — Perform a random job for limes.
  • lp!stockmarket — View all stocks and current prices.
  • lp!buy <stock> <amount> — Purchase stock shares.
  • lp!sell <stock> <amount> — Sell your shares.
  • lp!portfolio — View your current stocks.
  • lp!stockinfo <stock> — View specific stock details.
  • lp!markethistory — Show past stock price data.
  • lp!toplimes — Leaderboard of richest users.

👥 System Info & PluralKit

  • lp!system — Show your system info.
  • lp!whois <@user> — Get system info about a user.
  • lp!fronthistory — View fronting history.
  • lp!fronterinfo — See current fronter (if any).
  • lp!togglefronterinfo — Toggle automatic display of fronter info.
  • lp!clearfronterinfo — Clear saved fronter info for this server.
  • lp!setfrontnotify <#channel> — Set a channel to post front notifications.
  • lp!removefrontnotify — Disable front change notifications.
